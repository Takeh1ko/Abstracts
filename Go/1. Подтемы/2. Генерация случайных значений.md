# Генерация случайных чисел

```go
import (
    "math/rand"
    "time"
)

// Инициализация генератора
rand.Seed(time.Now().UnixNano())

// Случайное число от 0 до n-1
rand.Intn(100)  // 0-99

// Случайное число от 0.0 до 1.0
rand.Float64()

// Случайное число в диапазоне min-max
min := 10
max := 50
rand.Intn(max-min+1) + min

// Случайное число int64
rand.Int63()

// Случайная строка из символов
chars := "abcdefghijklmnopqrstuvwxyz"
result := make([]byte, 10)
for i := range result {
    result[i] = chars[rand.Intn(len(chars))]
}
```

Go 1.20+:
```go
import "math/rand"

// Автоматическая инициализация, seed не нужен
rand.Intn(100)
rand.Float64()
```

Локальный генератор:
```go
r := rand.New(rand.NewSource(time.Now().UnixNano()))
r.Intn(100)
```
