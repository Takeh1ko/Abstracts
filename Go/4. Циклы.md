# Циклы

## for - основной цикл

```go
// Классический цикл
for i := 0; i < 10; i++ {
    fmt.Println(i)
}

// Только условие (аналог while)
i := 0
for i < 10 {
    fmt.Println(i)
    i++
}

// Бесконечный цикл
for {
    fmt.Println("бесконечно")
    break // выход из цикла
}
```

## range - итерация по коллекциям

```go
// Массив/слайс
arr := []int{1, 2, 3}
for i, v := range arr {
    fmt.Printf("индекс: %d, значение: %d\n", i, v)
}

// Только значения
for _, v := range arr {
    fmt.Println(v)
}

// Только индексы
for i := range arr {
    fmt.Println(i)
}

// Строка (руны)
for i, r := range "привет" {
    fmt.Printf("%d: %c\n", i, r)
}

// Map
m := map[string]int{"a": 1, "b": 2}
for k, v := range m {
    fmt.Printf("%s: %d\n", k, v)
}

// Канал
ch := make(chan int)
for v := range ch { // читает до закрытия канала
    fmt.Println(v)
}
```

## Управление циклом

```go
for i := 0; i < 10; i++ {
    if i == 3 {
        continue // пропустить итерацию
    }
    if i == 7 {
        break // выйти из цикла
    }
    fmt.Println(i)
}
```

## Вложенные циклы с метками

```go
outer:
for i := 0; i < 3; i++ {
    for j := 0; j < 3; j++ {
        if i == 1 && j == 1 {
            break outer // выход из внешнего цикла
        }
        fmt.Printf("%d,%d ", i, j)
    }
}
```
