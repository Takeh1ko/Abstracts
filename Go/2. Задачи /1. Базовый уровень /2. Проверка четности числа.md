# Проверка четности числа

Напишите программу, которая определяет, является ли введенное целое число четным или нечетным.

## Входные данные

```
8
```

## Выходные данные

```
четное
```


# Решение
<details>
<summary>Solution (Click) </summary>

```go
package main

import "fmt"

func main() {
	num := 0
	fmt.Scan(&num)

	if num%2 == 0 {
		fmt.Println("четное")
	} else {
		fmt.Println("нечетное")
	}
}
```
